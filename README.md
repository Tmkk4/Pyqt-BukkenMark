# Pyqt-BukkenMark
It provides to compare the apartments you are considering renting.

# 使い方
  - 1. 気になっている物件のsuumo等の物件詳細ページのURLをフォームへ入力する
  - 2. 複数物件について以下の項目別で比較結果をみることができる
    - 家賃
    - 家賃と管理費・共益費の合計値
    - 敷金
    - 礼金
    - 初期費用
    - 間取り
    - 構造
    - 築年数
    - 専有面積
    - 主要採光面の向き
    - アクセス
    - 最寄り駅
